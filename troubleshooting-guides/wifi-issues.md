## üß≠ Step-by-Step Troubleshooting

### 1. Check Physical and Basic Conditions

* Confirm that Wi-Fi is enabled on the device
* Verify Airplane Mode is turned off
* Check if other devices can connect to the same network

**If multiple devices are affected:**
‚Üí The issue may be related to the router or internet service.

---

### 2. Restart Devices

* Restart the affected device
* Restart the modem and router (power off for 30 seconds)

This step resolves a large percentage of temporary connectivity issues.

---

### 3. Verify Network Settings

* Confirm the correct Wi-Fi network (SSID) is selected
* Re-enter the Wi-Fi password
* Check if the device is set to obtain IP address automatically (DHCP)

---

### 4. Check IP and Connectivity

* Verify if the device has a valid IP address
* Test connectivity using basic commands (if applicable):

  * `ping 8.8.8.8`
  * `ping google.com`

Results help determine whether the issue is local or DNS-related.

---

### 5. Signal Strength and Interference

* Check distance between device and router
* Avoid physical obstacles and interference
* If possible, test closer to the router

---

### 6. Update and Forget Network

* Forget the Wi-Fi network and reconnect
* Check for system or network driver updates

---

## üß© When to Escalate

Escalate the issue if:

* Router configuration changes are required
* Internet service appears to be down
* Hardware failure is suspected

Provide clear notes on steps already performed before escalation.
